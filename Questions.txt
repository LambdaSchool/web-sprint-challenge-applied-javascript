1. What is the DOM?

    A document object model that represents html on a page. This shows the format of said html and gives JS  a way to access it.

2. What is an event?

    An event is an action that is triggered by a conditional, which will in turn change something about the elements on the page.

3. What is an event listener?

    An event listener is a coding function that creates an event with a conditional.

4. Why would we convert a NodeList into an Array?

    So you can use array methods on them.

5. What is a component?

    A function  or part of a class that is reusable.
